Question-1: What are the different ways to select an element in the DOM?

Different ways to select an element in the DOM are: getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(),querySelectorAll(), getElementsByName()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The differences between innerHTML, innerText, and textContent are: innerHTML – Retrieves the entire content of an element, including HTML. innerText – Retrieves only the visible text of an element. textContent – Retrieves all text of an element, including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is attached to a parent element to handle events for its child elements. This helps in efficiently managing events, especially for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in the DOM where an event starts from the innermost element (the target element) and then propagates upwards to its parent elements. This allows parent elements to catch events triggered by their child elements.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, you use createElement() to create an element, appendChild() to add it, and remove() to remove it.
